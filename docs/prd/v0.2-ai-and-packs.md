# v0.2 тАФ AI, Skins, Collections & Pack spec (Released)

Goal: turn a file browser into a **workstation**. Give users a place
to bookmark stuff, reshape it with an assistant, and theme it to
their liking тАФ without a marketplace backend.

## Features

### Collections (Task 5)
- Persisted JSON at `<app_data>/collections.json`.
- Virtual path `collection:<id>` reuses the browsing stack; no
  bespoke view.
- Sidebar CRUD (create / rename / delete); Toolbar "Add toтАж" dropdown
  for the current selection.
- A file's appearance in many collections has zero duplication on
  disk.

### Archives (Task 8)
- `zip`, `tar`, `tar.gz` тАФ extract / create / list.
- Zip-slip hardened by canonicalising and prefix-checking every
  emitted path.
- Toolbar `Compress` / `Extract`; double-clicking an archive offers
  to unpack next to the file.

### Skins (Task 6) тАФ foundation of the Pack system
- 4 built-in: System Default, Ocean Dark, Nord, Solarized Light.
- A skin is a Pack with `skin.tokens: tokens.css`. The runtime
  injects it as `<style id="lb-skin-tokens">`, overriding CSS
  variables тАФ never component CSS.
- Picker at Sidebar footer (ЁЯОи Skins); install from a folder via
  `install_pack_from_folder`.

### AI Assistant (AI.md, 2026-05)
- OpenAI-compatible (`/v1/chat/completions` + function calling).
- 13 tools declared in [`src/ai/tools.ts`](../../src/ai/tools.ts),
  every one wraps an existing `api.*` call тАФ no AI-specific Rust
  command.
- AI Mode toggle flips a process-wide Rust `readonly` guard; while
  on, `move_to_trash` and `delete_forever` hard-fail. Scope is
  pinned to the current directory.
- Tool calls above `confirmThreshold` pause for Apply / Reject.
- Full technical deep-dive: [`AI.md`](../../AI.md).

### Pack spec (PACKS.md)
- One shape serves skins and plugins (`manifest.json` v1).
- `contributes.previewAdapters` / `contributes.archiveHandlers` /
  `contributes.aiTools` are declared but only the first reaches
  runtime in v0.3.

## Non-goals (v0.2)
- Marketplace UI / registry service.
- Native sidecar plugins (pushed to v0.4+).
- Multi-turn tool transaction / rollback (v0.4).

## Design notes
- Every AI write has to pass **four gates**: Rust guard, scope lock,
  confirm threshold, audit transcript. Not one relies on the model
  being trustworthy.
- The Pack file layout is intentionally human-editable: no build step,
  no lockfile, no JS runtime beyond dynamic import.
